## Current GHE.com Checks

When working with GHE.com, check the feature overview before making assumptions
about:

- public repository availability
- GitHub Marketplace and third-party Actions behavior
- API hostnames and URL formats
- GitHub Packages hostnames
- Actions OIDC issuer URLs
- raw and attachment URL formats
- feature preview availability
